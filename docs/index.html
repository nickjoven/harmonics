Proof chain A: Integers → Einstein

From energy conservation + stability, the Einstein field equations follow uniquely.

  1. Integers + fixed-point → $S^1$ (the circle)
  2. Energy conservation + stability → mediant (Stern-Brocot, 1858)
  3. Mediant enumeration → Stern-Brocot tree (configuration space)
  4. Self-consistency on the tree → rational field equation
  5. Mediant generators → $\mathrm{SL}(2,\mathbb{Z}) \to \mathrm{SL}(2,\mathbb{R})$, forcing $d = 3$
  6. Four conditions uniquely characterize $\mathrm{SL}(2,\mathbb{R})$ (Bianchi classification)
  7. Continuum limit at $K = 1$ → ADM evolution equations
  8. Lovelock uniqueness (1971) → $G_{\mu\nu} + \Lambda g_{\mu\nu} = 8\pi G\, T_{\mu\nu}$

Proof chain B: Integers → Schrödinger + Born

Same inputs, subcritical regime $K < 1$.

  1. Shared steps with Proof A (circle → SL(2) → $d = 3$)
  2. Parabola at tongue boundaries → $\Delta\theta \propto \sqrt{\varepsilon}$ (saddle-node universality)
  3. Basin measure with parabolic geometry → $P = |\psi|^2$
  4. $K < 1$ → partial locking; unlocked oscillators = quantum states
  5. Madelung variables $(\rho, S)$ for unlocked population
  6. Madelung ↔ Schrödinger equivalence → $i\hbar\, \partial_t \psi = H\psi$
