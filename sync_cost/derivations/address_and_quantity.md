# Address and Quantity

A ratio has two components. The numerator counts displacement.
The denominator counts period. One says how far; the other says
how long. These are not the same kind of information.

The mediant (a+c)/(b+d) does not divide. It locates: the unique
fraction between a/b and c/d with the smallest denominator. The
Stern-Brocot tree built from iterated mediants is an address book,
not an arithmetic table. Each node is a location in frequency space,
reached by a specific path of left and right steps from the root.

The four primitives operate on this pair:

| Primitive | Role |
|-----------|------|
| Integers | Counting (how many periods, how many windings) |
| Mediant | Locating (inserting an address between two others) |
| Fixed point | Self-reference (the address reads itself) |
| Parabola | Branching (the address has two sides: stable and unstable) |

The circle S¹ arises because the fixed-point condition identifies
all integer displacements: p ≡ 0 in the phase space, so R/Z = S¹.
Quantity (displacement) is taken modulo address (period). This
quotient is the circle.

The configuration surface has two dimensions because a fraction has
two components. The mediant operates on 2-vectors in SL(2,Z). One
dimension per component, each periodic. The Klein bottle and not
the torus because the two components are not interchangeable: the
denominator (period, temporal) carries the antiperiodic twist, the
numerator (displacement, spatial) does not. The asymmetry between
address and quantity is the asymmetry between time and space.

All derived numbers in the framework are either:

- **Addresses**: the integers {2, 3, 5, 8, 13, ...} that index
  modes on the Stern-Brocot tree. These are exact.

- **Quantities**: the rationals {3/2, 8/35, 27/8, 13/19, ...}
  built from addresses by arithmetic. These are exact.

- **Conversions**: the irrationals {φ, π, ln(3/2), C} that
  translate between the address book's discrete multiplicative
  structure and the continuous additive space where physics is
  measured. These are transcendental but determined.

- **Fixed points**: the self-consistent values {w*, K*, δ} that
  the address book computes about itself. These are determined to
  arbitrary precision by iteration.

The single dimensionful input (v = 246 GeV, or equivalently the
root oscillator's frequency in physical units) is the conversion
between the address book's intrinsic units and human units. It
says how large the universe's addresses are in meters and seconds.
The address book does not know meters. It knows only ratios.

This is prior to the four primitives. It is the structure of a
ratio: two components, one for where, one for how long. Everything
else — the circle, the tree, the Klein bottle, the three dimensions,
the gauge groups, the mass hierarchy, the cosmological constant —
is the self-consistent address book of a universe that can describe
itself.
