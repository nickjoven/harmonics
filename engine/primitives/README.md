# Layer 0: Primitives

## Claim

These four primitives — Z, Mediant, Fixed Point, Parabola — are hypothesized to be **necessary and sufficient** to derive the full structure of coupled oscillator dynamics.

## The Four Primitives

| Primitive | File | What it provides |
|---|---|---|
| Z | `z.md` | Counting, discreteness, iteration index |
| Mediant | `mediant.md` | The combining operation, betweenness, minimality |
| Fixed Point | `fixpoint.md` | Self-consistency, convergence, closure |
| Parabola | `parabola.md` | Nonlinearity, bifurcation, orientation, the Born exponent |

## Scientific Method Applied to Axioms

Each primitive is treated as a **hypothesis**, not a decree:

1. **Necessity test**: For each primitive, exhibit what fails without it. If removing a primitive leaves the remaining three sufficient, that primitive was not necessary and must be discarded.

2. **Sufficiency test**: Compose all four primitives and check whether the derived predictions match observation. If a prediction fails, at least one primitive is wrong or missing.

3. **No borrowed vocabulary**: Every term used in a definition is either (a) defined within these four files or (b) a logical connective (and, or, not, if-then, for-all, there-exists). No external mathematical definitions are imported.

## What This Is Not

- This is not a claim that these are "the axioms of mathematics."
- This is not a philosophical position about foundations.
- This is a concrete, testable hypothesis: these four, composed correctly, predict the observable structure of coupled oscillators — and nothing else is needed.
